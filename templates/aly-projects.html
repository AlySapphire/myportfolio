Princesses Vs Dragons

Princesses vs Dragons is a super cute fantasy series of multiplayer scenarios played as an over-the-shoulder third-person arena brawler for Steam, in which players are either a Dragon or a Princess competing over fairies. Each role has a unique set of skills they use in the battlefield in order to catch members of their opposing team. Drawn to the fairies that flit between flowers, both dragons and princesses clash in royal gardens, castle halls during festivals, and the floral d√©cor of dragon lairs.
My roles involved in this project included setting up a source control solution for team collaboration, programming the main character's mechanics, developing systems for integrating external data from CSV files into code and writing up technical documentation and specifications.

 Nuts Bolts & Goblins

This is a game I developed in Unity3D with TeamPoliteDynamite. Nuts Bolts and Goblins is a colorful and fast-moving rhythm style game where crude and nasty enemies can only be killed by using the correctly colored gun. I worked on the gameplay mechanics which involved shooting, weapon swapping, health and movement as well as the boss mechanics which included AI programming, state machine, and health. I also created and implemented an alternative to Unity's physics engine for a crucial game function.

Languages/Engines Used: C#, Unity3D

 DynamicSoundtrackSystem

The DynamicSoundtrackSystem is a C++ library, created by me, which gives developers an easy way to manipulate audio data to create a dynamic soundtrack. From using simple volume and mute controls, to advanced DSP filters with custom parameters, developers are able to utilize this library to create a vast number of effects that will greatly add to the immersion and enjoyment of any game.

Languages/API's Used: C++, FMOD

 SpaceJam

This simple but appealing rhythm game was made with UE4 in less 24 hours as part of a game jam. The player controls an innocent but musically aware slug who is just sliding through life. The slug must avoid various obstacles that appear to the beat of the in-game music, by jumping up to the ceiling and falling to the floor. I worked on the movement and gravity flipping mechanics as well as collision detection and adding in the animations.

Languages/Engines Used: UE4, Blueprints

 OpenGL Demo

This is a small demo application that showcases some OpenGL techniques and graphics concepts I researched and implemented. Some techniques shown in this application are:

  • Surface-based lighting with ambience and specular reflections
  • GPU-Accelerated particle system
  • Post-Processing effects such as Distortion and Sepia toning
  • Camera-frustum culling and loading an FBX model and animations and rendering them in 3D space using code.
Languages Used: C++, GLSL
